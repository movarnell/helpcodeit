Key Terms for HTML

  • Element: The basic building blocks of HTML.
  • Tag: Markup code that defines elements and attributes. Example: <p>This is a paragraph.</p>
  • Attribute: Additional information about an element, defined within the tag. Example: <a href="https://www.example.com">Link</a>
  • HTML Document Structure: The arrangement of <!DOCTYPE>, <html>, <head>, and <body> tags. Example: <!DOCTYPE html><html><head></head><body></body></html>
  • Meta Tags: Information about the HTML document, placed within <head>. Example: <meta name="description" content="This is a description">
  • Hyperlink: An element that links to another resource, typically using the <a> tag. Example: <a href="https://www.example.com">Link</a>
  • Text Formatting: Elements like <b>, <i>, and <strong> to format text. Example: <b>Bold text</b>
  • List: Ordered (<ol>) or unordered (<ul>) lists. Example: <ul><li>Item 1</li><li>Item 2</li></ul>
  • Table: Represents tabular data, made up of <table>, <tr>, <td>, and optionally <th> tags. Example: <table><tr><th>Header 1</th><th>Header 2</th></tr><tr><td>Row 1, Column 1</td><td>Row 1, Column 2</td></tr></table>
  • Form: Elements such as <form>, <input>, and <textarea> for user input. Example: <form action="/submit" method="post"><input type="text" name="username"></form>
  • CSS (Cascading Style Sheets): Used for styling HTML elements. Example: <style>p { color: red; }</style>
  • ID: Unique identifier for an element. Example: <div id="example"></div>
  • Class: used to apply a style to an individual item but made in a way that is reusable. Example: <p class="example">This is an example.</p>
  • Inline vs Block Elements: Elements that take up the full width (block) or only the necessary width (inline). Example: <div><p>This is a paragraph.</p></div>
  • Semantic Tags: Elements like <header>, <footer>, <article>, and <section> that carry meaning. Example: <header><h1>Page Title</h1></header>
  • Media Elements: <img>, <audio>, <video> for embedding media. Example: <img src="image.jpg" alt="An image">
  • Accessibility: ARIA roles, alt attributes, and other practices to make web pages more accessible. Example: <img src="image.jpg" alt="An image">
  • DOCTYPE: Declares the document type and version of HTML. Example: <!DOCTYPE html>
  • Viewport: Meta tag for responsive design. Example: <meta name="viewport" content="width=device-width, initial-scale=1.0">
  • HTML Entities: Special characters represented in HTML, like &amp; for &. Example: <p>This is an example &ndash; a dash.</p>
  • Comments: Code ignored by browsers, used for notes (<!-- Comment -->). Example: <!-- This is a comment -->
