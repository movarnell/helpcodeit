Key Terms for CSS

  • Selector

    A selector is an element that specifies which elements in a document the CSS rule applies to. Selectors can be based on element names, classes, IDs, attributes, and more.

    
/* Selects all 
     elements */
p {
  color: red;
}

/* Selects all elements with class "example" */
.example {
  color: blue;
}

/* Selects the element with ID "main" */
#main {
  color: green;
}

  • Property

    A property is a specific characteristic of an element that can be styled with CSS. Properties include things like color, font-size, and margin.

    
/* Sets the color of all 
     elements to red */
p {
  color: red;
}

/* Sets the font-size of all elements with class "example" to 16px */
.example {
  font-size: 16px;
}

/* Sets the margin of the element with ID "main" to 20px */
#main {
  margin: 20px;
}

  • Value

    A value is the specific setting for a CSS property. For example, the value for the color property could be "red", "blue", "green", or any other valid color value.

    
/* Sets the color of all 
     elements to red */
p {
  color: red;
}

/* Sets the font-size of all elements with class "example" to 16px */
.example {
  font-size: 16px;
}

/* Sets the margin of the element with ID "main" to 20px */
#main {
  margin: 20px;
}

  • Declaration

    A declaration is a combination of a property and a value. Declarations are used to specify the styles for a particular selector.

    
/* Sets the color and font-size of all 
     elements */
p {
  color: red;
  font-size: 16px;
}

/* Sets the margin and padding of the element with ID "main" */
#main {
  margin: 20px;
  padding: 10px;
}

  • Cascade

    The cascade is the process by which CSS determines which styles to apply to an element when there are conflicting styles. The cascade takes into account the specificity of the selectors, the order in which the styles are defined, and whether or not the styles are marked as !important.

    
/* This style will be applied to all 
     elements */
p {
  color: red;
}

/* This style will be applied to all elements with class "example" */
.example {
  color: blue;
}

/* This style will be applied to the element with ID "main" */
#main {
  color: green;
}

/* This style will be applied to the element with ID "main" because it is marked as !important */
#main {
  color: yellow !important;
}

Common CSS Attributes

This list is not comprehensive but just serves as a example of some of the most commonly used attributes. For a more complete list check out the W3 Schools Page for CSS Reference

Attribute Description Example
color Sets the color of text This text is red
font-size Sets the size of text This text is 24px
background-color Sets the background color of an element This has a yellow background
border Sets the border of an element This has a black border
padding Sets the padding of an element. Padding is the area between the content and the border. This has 10px of padding
margin Sets the margin of an element. Margin is the area outside the border. This has 10px of margin
