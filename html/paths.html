Path Issues

"It's like loosing your keys, in the couch. You know where they are, but do you really?"

What is a path? A path is a way to get to a file or folder on your computer. It is a series of folders that you have to go through to get to the file or folder you want. For example, if you want to get to a file called "index.html" that is in a folder called "html" that is in a folder called "helpcodeit" that is in a folder called "Documents" that is in a folder called "Users" that is in a folder called "C:" that is in a folder called "Computer" that is in a folder called "This PC"...Confused yet? Don't worry, I am too. Let's look at a diagram to help us understand this better.

The "./" in the diagram above is the current folder. If you used "../" that would take you to the parent folder, aka up one level. The parent folder is the folder above the current folder. The parent folder is the folder that contains the current folder. The parent folder is the folder that the current folder is in. The parent folder is the folder that the current folder is a subfolder of. The parent folder is the folder that the current folder is a child folder of.

Absolute Paths

An absolute path is a path that starts from the root folder. The root folder is the top folder in the hierarchy. In the diagram above, the root folder is "This PC". An absolute path is the full path to a file or folder. For example, the absolute path to the file "index.html" in the diagram above is "C:\Users\Documents\helpcodeit\html\index.html".

Think of the hierarchy as a pyramid. The root folder is at the top of the pyramid. The folders below the root folder are the next level of the pyramid. The folders below those folders are the next level of the pyramid. And so on. The file or folder you want to get to is at the bottom of the pyramid. The absolute path is the path from the top of the pyramid to the bottom of the pyramid.

Most of the time we only need to get from the file we are writing to the file we want to link to. We don't need to go all the way to the top of the pyramid. We just need to go up a few levels. We can do this with a relative path which is shown in the illustration at the top of the page.
