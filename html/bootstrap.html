Using Bootstrap

Bootstrap is a CSS framework that makes it easy to create responsive websites. It is a great tool for beginners to use to create a website that looks good on all devices.

There are some things that are useful to know when installing bootstrap into your project. First off there are two different ways to link your site to bootstrap.

CDN

CDN stands for Content Delivery Network. This is a way to link your site to a file that is hosted on a server somewhere else. This is a great way to link to bootstrap because it is easy to do and it is fast. The downside to this is that if the server goes down or if the file is deleted then your site will not be able to access the file and your site will not work properly. To link to bootstrap using a CDN you can use the following code:

<!-- CSS only -->
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/css/bootstrap.min.css" rel="stylesheet" integrity="sha384-T3c6CoIi6uLrA9TneNEoa7RxnatzjcDSCmG1MXxSR1GAsXEV/Dwwykc2MPK8M2HN" crossorigin="anonymous" crossorigin="anonymous">

<!-- JavaScript Bundle with Popper -->
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/js/bootstrap.bundle.min.js" integrity="sha384-C6RzsynM9kWDrMNeT87bh95OGNyZPhcTNXj1NW7RuBCsyN/o0jlpcV8Qyq46cDfL" crossorigin="anonymous"></script>

Download

The other way to link to bootstrap is to download the files and link to them locally. This is a great way to link to bootstrap because you will always have access to the files and your site will always work. The downside to this is that it is slower than using a CDN. To install the latest version of bootstrap to your project using npm you can use the following code:

npm install bootstrap
To link to bootstrap using a download you can use the following code:

<!-- CSS only -->
<link href="./node_modules/bootstrap/dist/css/bootstrap.min.css" rel="stylesheet" crossorigin="anonymous">

<!-- JavaScript Bundle with Popper -->
<script src="node_modules/bootstrap/dist/js/bootstrap.bundle.min.js" crossorigin="anonymous"></script>

Example of Linking using CDN

Here is an example of how to link to bootstrap using a CDN.
If linked correctly the button below should be blue on your html page when opened.

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <link href="https://cdn.jsdelivr.net/npm/bootstrap/dist/css/bootstrap.min.css" rel="stylesheet" crossorigin="anonymous">
  <title>Document</title>
</head>
<body>
  <h1>Hello World</h1>
  <button type="button" class="btn btn-primary">Primary</button>
  
  <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/js/bootstrap.bundle.min.js" integrity="sha384-C6RzsynM9kWDrMNeT87bh95OGNyZPhcTNXj1NW7RuBCsyN/o0jlpcV8Qyq46cDfL" crossorigin="anonymous"></script>
</body>
</html>
