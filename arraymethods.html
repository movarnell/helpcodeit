Array Methods

Method Name Return Value Explanation Example
push() The modified array Appends one or more elements to the end of the array. let array = [];
array.push("hello", "world");
["hello", "world"]
pop() The removed element Removes the last element from the array and returns it. let array = [ 1, 2, 3]
const lastElement = arr.pop();
[3]
shift() The removed element Removes the first element from the array and returns it. let array = [1,2,3];
const firstElement = array.shift();
[1]
unshift() The modified array Prepends one or more elements to the beginning of the array. let array =[1,2,3];
array.unshift("hello", "world");
["hello", "world", 1, 2, 3]
splice() A new array with the removed elements Removes elements from the array and optionally inserts new elements at the specified indexes. let array = ["hello", "world", "welcome", "home"]
const newArray = array.splice(1, 2, "hello", "world");
["world", "welcome"]
slice() A new array with the specified elements Copies a section of the array and returns it. let array = ["hello", "world", "welcome", "home"]
const newArray = array.slice(1, 3);
["world", "welcome"]
concat() A new array with the concatenated arrays Concatenates two or more arrays and returns a new array. let array = ["hello", "world", "welcome", "home"]
const newArray = array.concat(["hello", "world"]);
["hello", "world", "welcome", "home", "hello", "world"]
join() A string with the elements of the array Joins the elements of the array into a string. let array = ["hello", "world", "welcome", "home"]
const newString = array.join(" ");
"hello world welcome home"
map() A new array with the results of the function Applies a function to each element of the array and returns a new array with the results. let array = [1, 2, 3, 4, 5];
const newArray = array.map(element => element * 2);
[2, 4, 6, 8, 10]
filter() A new array with the elements that pass the test Creates a new array with the elements of the original array that pass a test. let array = [1, 2, 3, 4, 5];
const newArray = array.filter(element => element % 2 === 0);
[2, 4]
reduce() The accumulated value Reduces the array to a single value by applying a function to each element and accumulating the results. let array = [1, 2, 3, 4, 5];
const sum = array.reduce((accumulator, element) => accumulator + element, 0);
15
reduceRight() The accumulated value Reduces the array to a single value by applying a function to each element in reverse order and accumulating the results. let array = [1, 2, 3, 4, 5];
const sum = array.reduceRight((accumulator, element) => accumulator + element, 0);
15
every() True or false Tests whether all elements in the array pass the test implemented by the provided function. let array = [1, 2, 3, 4, 5];
const allEven = array.every(element => element % 2 === 0);
false
some() True or false Tests whether some element in the array passes the test implemented by the provided function. let array = [1, 2, 3, 4, 5];
const someEven = array.some(element => element % 2 === 0);
true
includes() True or false Tests whether the array contains the specified element. let array = [1, 2, 3, 4, 5];
const hasTwo = array.includes(2);
true
indexOf() The index of the element Returns the first index of the specified element in the array. let array = [1, 2, 3, 4, 5]
const index = array.indexOf(2);
1
lastIndexOf() The index of the element Returns the last index of the specified element in the array. let array = [1, 2, 3, 4, 5]
const index = array.lastIndexOf(2);
1
find() The value of the element Returns the value of the first element in the array that satisfies the provided testing function. let array = [1, 2, 3, 4, 5];
const element = array.find(element => element % 2 === 0);
2
findIndex() The index of the element Returns the index of the first element in the array that satisfies the provided testing function. let array = [1, 2, 3, 4, 5];
const index = array.findIndex(element => element % 2 === 0);
1
sort() The modified array Sorts the elements of the array in place. let array = ["e", "r" , "d", "v", "a"];
array.sort();
["a", "d", "e", "r", "v"]
reverse() The modified array Reverses the order of the elements of the array in place. let array = [1, 2, 3, 4, 5];
array.reverse();
[5, 4, 3, 2, 1]
fill() The modified array Fills all the elements of the array from a start index to an end index with a static value. let array = [1, 2, 3, 4, 5];
array.fill(0, 1, 3);
[1, 0, 0, 4, 5]
copyWithin() The modified array Copies a sequence of array elements within the array. let array = [1, 2, 3, 4, 5];
array.copyWithin(0, 3);
[4, 5, 3, 4, 5]
flat() A new array with the sub-array elements concatenated Creates a new array with all sub-array elements concatenated into it recursively up to the specified depth. const arr = [1, 2, [3, 4]];
arr.flat();
[1, 2, 3, 4]
flatMap() A new array with the sub-array elements concatenated Maps each element using a mapping function, then flattens the result into a new array. const arr = [1, 2, 3, 4];
arr.flatMap(x => [x * 2]);
[2, 4, 6, 8]
