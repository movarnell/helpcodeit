Sample Page

This page is an example template

When you build out your page feel free to put anything you like in the main content area here, but make sure you don't get rid of the navbar link, and that you name your HTML file with your name. If you choose to use your own CSS you will need to add it as a link in the header. All your files except images will go in the community folder. images go in the image folder, and you will need to check the path for the source image.
