Installing VS Code

VS Code is a free, open source code editor that is lightweight like a text editor, but offers many of the same features as an IDE (integrated development environment). It's a great tool for beginners and experts alike, and is the most popular code editor in the world.

VS Code is available for Windows, Mac, and Linux. You can download it here: https://code.visualstudio.com/download

Once you've downloaded the installer, run it and follow the instructions to install VS Code on your computer.

There are thousands of extensions you can use with VS Code, but to start off we only recommend the 'Live Server' extension by Ritwick Dey.

It is easy to get extensions that conflict with each other. If you install something that doesn't work with what your doing you can either disable or uninstall the extension.
