Key Terms for React Development

  • React

    A JavaScript library for building user interfaces. It is common in React development to need to import components that are reused. Below is an example of an import. 

    import React from 'react';

  • JSX

    A syntax extension for JavaScript that allows you to write HTML-like code in your JavaScript files. The most commonly seen change is having to change 'class' to 'className' like the example below

    <h1 className="display-1">Hello, world!</h1>;

  • Component

    A reusable piece of code that defines a part of a user interface. Typically anything that may need to be reused in a project is made into its own component.
