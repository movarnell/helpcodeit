Key Terms for JavaScript

  • Variable: A container for a value, defined with let or const.

    let myVariable = "Hello, world!";
  • String: A sequence of characters, defined with single or double quotes.

    let myString = "Hello, world!";
  • Number: A numeric value, defined with or without decimals.

    let myNumber = 42;
  • Boolean: A value that is either true or false.

    let myBoolean = true;
  • Array: A list of values, defined with square brackets.

    let myArray = [1, 2, 3];
  • Object: A collection of key-value pairs, defined with curly braces. For example, let's say we want to store information about a person. We can create an object with properties like name and age:

    let person = {name: "John", age: 30};
  • Function: A reusable block of code, defined with the function keyword. For example, let's say we want to create a function that prints "Hello, world!" to the console. We can define the function like this:

    function sayHello() {console.log("Hello, world!");}
  • Method: A function that is a property of an object. For example, let's say we have an object called person and we want to add a method to greet someone. We can define the method like this:

    let person = {name: "John", age: 30, greet: function() {console.log("Hello, world!");}};
  • Operator: A symbol that performs an operation on one or more values. For example, let's say we want to add two numbers together. We can use the + operator like this:

    let sum = 1 + 2;
  • Comparison Operator: An operator that compares two values and returns a boolean. For example, let's say we want to check if two values are equal. We can use the === operator like this:

    let isEqual = 1 === 1;
  • Conditional Statement: A statement that runs different code depending on a condition. For example, let's say we want to check if a number is greater than 10. We can use an if statement like this:

    let x = 5; if (x > 10) {console.log("x is greater than 10");} else {console.log("x is less than or equal to 10");}
  • Logical Operator: An operator that combines multiple boolean values. Examples include &&, ||, and !. For example, let's say we want to check if two conditions are true. We can use the && operator like this:

    let isTrue = true && false;
  • Loop: A statement that runs code repeatedly. For example, let's say we want to print the numbers 0 to 9 to the console. We can use a for loop like this:

    for (let i = 0; i < 10; i++) {console.log(i);}
  • Event: An action that occurs in the browser, such as a click or keypress. For example, let's say we want to listen for a click event on a button. We can use the addEventListener method like this:

    document.addEventListener("click", function() {console.log("Clicked!");});
  • Event Listener: A function that runs when a specific event occurs. For example, let's say we want to listen for a click event on a button and run a function when it happens. We can define the function and add it as an event listener like this:

    document.addEventListener("click", handleClick); function handleClick() {console.log("Clicked!");}
  • DOM (Document Object Model): A representation of the HTML document, which can be modified with JavaScript. For example, let's say we want to get an element with the ID "my-element". We can use the getElementById method like this:

    let myElement = document.getElementById("my-element");
  • DOM Node: An element, attribute, or text in the DOM. For example, let's say we want to get an element with the ID "my-element". The element itself is a DOM node:

    let myElement = document.getElementById("my-element");
  • DOM Query: A method that returns a DOM node. For example, let's say we want to get an element with the ID "my-element". We can use the getElementById method like this:

    let myElement = document.getElementById("my-element");
  • DOM Manipulation: Changing the DOM with JavaScript. For example, let's say we want to change the text inside an element with the ID "my-element". We can use the innerHTML property like this:

    document.getElementById("my-element").innerHTML = "Hello, world!";
  • DOM Event: An event that occurs on a DOM node. For example, let's say we want to listen for a click event on a button. We can use the addEventListener method like this:

    document.addEventListener("click", function() {console.log("Clicked!");});
  • DOM Event Listener: A function that runs when a DOM event occurs. For example, let's say we want to listen for a click event on a button and run a function when it happens. We can define the function and add it as an event listener like this:

    document.addEventListener("click", handleClick); function handleClick() {console.log("Clicked!");}
  • DOM Event Handler: A function that runs when a DOM event occurs. For example, let's say we want to listen for a click event on a button and run a function when it happens. We can define the function and add it as an event listener like this:

    document.addEventListener("click", handleClick); function handleClick() {console.log("Clicked!");}
  • DOM Event Propagation: The order in which event handlers are triggered. For example, let's say we have a button inside a div, and both have click event listeners. When we click the button, the button's event listener is triggered first, followed by the div's event listener. We can stop the event from propagating to the div by calling the stopPropagation method:

    event.stopPropagation();
  • DOM Event Bubbling: Event handlers on child elements are triggered before event handlers on parent elements. For example, let's say we have a button inside a div, and both have click event listeners. When we click the button, the button's event listener is triggered first, followed by the div's event listener.

    document.addEventListener("click", function() {console.log("Clicked!");});
  • DOM Event Capturing: Event handlers on parent elements are triggered before event handlers on child elements. For example, let's say we have a button inside a div, and both have click event listeners. When we click the button, the div's event listener is triggered first, followed by the button's event listener. We can enable capturing by passing a third argument to addEventListener with the value true:

    document.addEventListener("click", function() {console.log("Clicked!");}, true);
  • DOM Event Delegation: Attaching a single event listener to a parent element to handle events on child elements. For example, let's say we have a list of items, and we want to listen for clicks on each item. Instead of adding a separate event listener to each item, we can add a single event listener to the list itself and use the event target to determine which item was clicked:

    document.addEventListener("click", function(event) {if (event.target.matches(".my-class")) {console.log("Clicked!");}});
  • DOM Event Object: An object that contains information about a DOM event. For example, let's say we want to log the target of a click event. We can access the target property of the event object:

    document.addEventListener("click", function(event) {console.log(event.target);});
  • DOM Event Prevent Default: A method that prevents the default behavior of an event. For example, let's say we have a link, and we want to prevent it from navigating to a new page when clicked. We can call the preventDefault method on the event object:

    event.preventDefault();
