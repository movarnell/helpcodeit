Array methods are built-in JavaScript methods that allow us to perform actions on arrays. These methods are extremely useful and can help us write more efficient code.

Below is a list of useful array functions with examples of how they work. Use the table of contents to go to the example, then at the example you can click on the method name and go to the W3 Schools page for that array method, with interactive examples.

  • fill()

    Fills all the elements of the array from a start index to an end index with a static value. The end index is not included. If the start index is negative, the start index is determined from the end of the array. The first number is the value to fill the array with, the second number is the start index, and the third number is the end index.

    let array = [1, 2, 3, 4, 5];
      array.fill(0, 1, 3);
      [1, 0, 0, 4, 5]

  • copyWithin()

    Copies a sequence of array elements within the array.

              array.copyWithin(0, 3);

  • flat()

    Creates a new array with all sub-array elements concatenated into it recursively up to the specified depth.

    let array = [1, 2, [3, 4, [5, 6]]];
              const newArray = array.flat(2);
              [1, 2, 3, 4, 5, 6]
  • flatMap()

    Maps each element using a mapping function, then flattens the result into a new array.

    const arr = [1, 2, 3, 4];
              arr.flatMap(x => [x * 2]);
              [2, 4, 6, 8]
  • slice()

    Returns a shallow copy of a portion of an array into a new array object selected from begin to end (end not included).

    let array = [1, 2, 3, 4, 5];
              const newArray = array.slice(1, 3);
              [2, 3]
  • splice()

    Changes the contents of an array by removing or replacing existing elements and/or adding new elements.

    let array = [1, 2, 3, 4, 5];
              array.splice(2, 1, "a", "b");
              [3]
  • concat()

    Merges two or more arrays into a new array.

    let array1 = [1, 2, 3];
              let array2 = [4, 5, 6];
              let newArray = array1.concat(array2);
              [1, 2, 3, 4, 5, 6]
  • join()

    Joins all elements of an array into a string.

    let array = [1, 2, 3, 4, 5];
              const string = array.join("-");
              "1-2-3-4-5"
  • toString()

    Returns a string representing the specified array and its elements.

    let array = [1, 2, 3, 4, 5];
              const string = array.toString();
              "1,2,3,4,5"
  • length

    Reflects the number of elements in an array.

    let array = [1, 2, 3, 4, 5];
              array.length;
              5
  • pop()

    Removes the last element from an array and returns that element.

    let array = [1, 2, 3, 4, 5];
              array.pop();
              [1, 2, 3, 4]
  • push()

    Adds one or more elements to the end of an array and returns the new length of the array.

    let array = [1, 2, 3, 4, 5];
              array.push(6);
              [1, 2, 3, 4, 5, 6]
  • shift()

    Removes the first element from an array and returns that element.

    let array = [1, 2, 3, 4, 5];
              array.shift();
              [2, 3, 4, 5]
  • unshift()

    Adds one or more elements to the beginning of an array and returns the new length of the array.

    let array = [1, 2, 3, 4, 5];
              array.unshift(0);
              [0, 1, 2, 3, 4, 5]
