Understanding Variables in JavaScript

What is a Variable?

A variable is like a box where we can keep things. In programming, we use variables to store information like numbers, words, or more complex stuff.

Using 'var'

The 'var' way to make a variable is the old way. We don't use it anymore, but you might still see it. Here's an example:

var myName = 'Alice';

Using 'let'

'let' is the new way to make a variable that you can change later. Here's an example:

let myAge = 10;
myAge = 11; // You can change it!

Using 'const'

If you want to make a variable that never changes, use 'const'. Here's how:

const mySchool = 'Cool School';

Summary

Use 'var' if you see old code, 'let' if you want to change the variable, and 'const' if you never want it to change. Happy coding!
