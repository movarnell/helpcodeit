Strings are a primitive data type in JavaScript. They are used to represent text and are wrapped in single or double quotes. Strings are immutable, meaning they cannot be changed once they are created. However, you can use string methods to manipulate strings and return new values.

  • charAt()

    Returns the character at the specified index in a string.

    let str = "hello";
          const char = str.charAt(1);
          "e"
  • charCodeAt()

    Returns the Unicode value of the character at the specified index in a string.

    let str = "hello";
          const charCode = str.charCodeAt(1);
          101
  • concat()

    Joins two or more strings and returns a new string.

    let str1 = "Hello";
          let str2 = "World";
          const newStr = str1.concat(" ", str2);
          "Hello World"
  • endsWith()

    Checks whether a string ends with a specified string/characters.

    let str = "Hello World";
          const endsWith = str.endsWith("World");
          true
  • includes()

    Checks whether a string contains the specified string/characters.

    let str = "Hello World";
          const includes = str.includes("World");
          true
  • indexOf()

    Returns the position of the first occurrence of a specified value in a string.

    let str = "Hello World";
          const index = str.indexOf("World");
          6
  • lastIndexOf()

    Returns the position of the last occurrence of a specified value in a string.

    let str = "Hello World";
          const lastIndex = str.lastIndexOf("l");
          9
  • localeCompare()

    Compares two strings in the current locale.

    let str1 = "Hello";
          let str2 = "hello";
          const compare = str1.localeCompare(str2);
          -1
  • match()

    Searches a string for a match against a regular expression, and returns the matches.

    let str = "The rain in Spain falls mainly in the plain";
          const regex = /ain/g;
          const matches = str.match(regex);
          ["ain", "ain", "ain"]
  • repeat()

    Returns a new string with a specified number of copies of an existing string.

    let str = "Hello";
          const newStr = str.repeat(3);
          "HelloHelloHello"
  • replace()

    Searches a string for a specified value, or a regular expression, and returns a new string where the specified values are replaced.

    let str = "Hello World";
          const newStr = str.replace("World", "Universe");
          "Hello Universe"
  • search()

    Searches a string for a specified value, or regular expression, and returns the position of the match.

    let str = "Hello World";
          const index = str.search("World");
          6
  • slice()

    Extracts a part of a string and returns a new string.

    let str = "Hello World";
          const newStr = str.slice(0, 5);
          "Hello"
  • split()

    Splits a string into an array of substrings.

    let str = "Hello World";
          const arr = str.split(" ");
          ["Hello", "World"]
  • startsWith()

    Checks whether a string starts with a specified string/characters.

    let str = "Hello World";
          const startsWith = str.startsWith("Hello");
          true
  • substr()

    Extracts a part of a string, beginning at a specified start position, and returns a specified number of characters.

    let str = "Hello World";
          const newStr = str.substr(6, 5);
          "World"
  • substring()

    Extracts the characters from a string, between two specified indices.

    let str = "Hello World";
          const newStr = str.substring(6, 11);
          "World"
  • toLocaleLowerCase()

    Converts a string to lowercase letters, according to the host's locale.

    let str = "Hello World";
          const newStr = str.toLocaleLowerCase();
          "hello world"
  • toLocaleUpperCase()

    Converts a string to uppercase letters, according to the host's locale.

    let str = "Hello World";
          const newStr = str.toLocaleUpperCase();
          "HELLO WORLD"
  • toLowerCase()

    Converts a string to lowercase letters.

    let str = "Hello World";
          const newStr = str.toLowerCase();
          "hello world"
  • toUpperCase()

    Converts a string to uppercase letters.

    let str = "Hello World";
          const newStr = str.toUpperCase();
          "HELLO WORLD"
  • toString()

    Returns the value of a string.

    let str = "Hello World";
          const value = str.toString();
          "Hello World"
  • trim()

    Removes whitespace from both ends of a string.

    let str = "   Hello World   ";
          const newStr = str.trim();
          "Hello World"
  • valueOf()

    Returns the primitive value of a string.

    let str = "Hello World";
          const value = str.valueOf();
          "Hello World"
