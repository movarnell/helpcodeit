Contribute to Our Open Source Project

We are excited that you are interested in contributing to our project. The main contributions we need at this time are 'student pages' in the Community Pages section.

After the video proceed to the contribution instructions below.

Follow these steps to fork our repository, make a new branch, and create a pull request:

1. Forking a Repository

Click the button below to go to the repository, then click the 'Fork' button at the top right of the page.

Go toRepository

2. Cloning Your Fork

On your forked repository page, click the 'Code' button, copy the URL, and then use git clone "Your URL" in your terminal.

3. Creating a New Branch

Use git checkout -b branch-name to create and switch to a new branch.Please name the new branch with your first initial and last name. For example, if your name is John Smith, your branch name would be 'jSmith'. This will make it easier to keep track of who is working on what. You can push your code back to your cloned fork at any time without affecting the main project.

4. Making Changes

Make changes to the files using your favorite code editor. Add your HTML page to the community folder, and make sure you put your link (with your name) in the community.html file. Make sure you use the sample student page as a template, as it is a requirment that we keep the navbar at the top of the page. Anything else on your page is yours to customize as you please. No inappropriate content will be accepted. If you have any questions about what is appropriate, please contact us. Feel free to pass on your experiences and tips for the future students. We want to make this a great resource for future students. Links to your favorite resources are also welcome and encouraged.

5. Committing Changes

Add and commit your changes using git add . and git commit -m "description".

6. Pushing Changes to GitHub

Push your changes with git push origin branch-name.

7. Creating a Pull Request

Go to your fork on GitHub, click 'New Pull Request', select your branch, fill out the form, and click 'Create Pull Request'.

Thanks for contributing to our project! If you have any questions, please contact us. Make sure when your finished you have also added your name to the contributors list on the readme so you can get credit for your work. This is a project you can happily add to your portfolio and put on your resume. We look forward to seeing your work!
