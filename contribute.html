Contribute to Our Open Source Project

Follow these steps to fork our repository, make a new branch, and create a pull request:

1. Forking a Repository

Click the button below to go to the repository, then click the 'Fork' button at the top right of the page.

Go toRepository

2. Cloning Your Fork

On your forked repository page, click the 'Code' button, copy the URL, and then use git clone URL in your terminal.

3. Creating a New Branch

Use git checkout -b branch-name to create and switch to a new branch.Please name the new branch with your first initial and last name. For example, if your name is John Smith, your branch name would be 'jSmith'. This will make it easier to keep track of who is working on what.

4. Making Changes

Make changes to the files using your favorite code editor. To find what you can work on please look at the issues on github.

See Issues

5. Committing Changes

Add and commit your changes using git add . and git commit -m "description".

6. Pushing Changes to GitHub

Push your changes with git push origin branch-name.

7. Creating a Pull Request

Go to your fork on GitHub, click 'New Pull Request', select your branch, fill out the form, and click 'Create Pull Request'.

Thanks for contributing to our project! If you have any questions, please contact us.
